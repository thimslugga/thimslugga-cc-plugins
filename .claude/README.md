## CLAUDE.md

`CLAUDE.md` is context, not hard enforcement. It is delivered as a strong instruction and Claude generally follows it, but it is a behavioral nudge rather than a guaranteed rule. Using concrete bullet phrasing will gives it some teeth. It also only governs Claude Code. It does not touch Claude.ai or the API, which have their own settings, so you would set the equivalent there separately if you want it everywhere.
